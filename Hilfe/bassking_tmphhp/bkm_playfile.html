BKM_PlayFile
 Zurück Top Weiter
Startet die Wiedergabe einer Datei.

Verwenden Sie BKM_Play um die Wiedergabe der aktuell ausgewählten Datei zu starten.
sParam1 muss den absoluten Pfad zur Datei oder eine gültige Internet Radio-URL beinhalten.

Syntax
BassKing_SendMessage(#BKM_PlayFile, NULL, NULL, sParam1)
