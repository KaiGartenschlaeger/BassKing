Registriert das Plugin bei BassKing.

Dieser Befehl ist nötig, damit BassKing das Plugin mit Nachrichten versorgt.
Ohne dieser Registrierung, erhält das Plugin keine Nachrichten von BassKing, und kann somit nicht auf bestimmten Ereignissen reagieren.
Dieser Befehl sollte als erstes aufgerufen werden

Syntax
BassKing_SendMessage(#BKM_Register)
