Befehleübersicht;


BKM_Register

Mit diesem Befehl wird das Plugin bei BassKing eingetragem, somit erhält das Plugin sämtliche Informatioen von BassKing.
Ohne dieser Nachricht, erhällt das Plugin z.B. keine Titelinformationen.
Dieser Befehl sollte als erstes aufgerufen werden!

Rückgabewert
1 falls erfolgreich, andernfalls 0

BKM_Unregister

Hiermit wird das Plugin abgemeldet.

Rückgabewert
keiner

BKM_End

Beendet BassKing

Rückgabewert
keiner
